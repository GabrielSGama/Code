Descrição:
Este programa é um simulador de matriz esparsa.
A linguagem utilizada foi c e o programa consiste em: criar/remover matrizes e adicionar/modificar/visualizar os seus elementos.

Compilar e executar:
Para Linux-> Abra o terminal e acesse o diretório onde está o arquivo .c. Em seguida, digite: # gcc matriz.c -o "nome desejado".out
Após compilar o programa, poderá executá-lo com o comando: ./"nome desejado".out

Para Windows-> Pode-se utilizar o auxílio de IDEs, como o Code:Blocks, para compilar e executar o programa.
